# pycmplot benchmark suite

Benchmarks pycmplot against CMplot (R), gwaslab, qmplot, and qqman across
dataset sizes of 500K-10M variants on a single SLURM compute node.

## Directory structure

```
|--benchmark/
   |--generate_sumstats.py   # Synthetic GWAS data generator
   |--bench_python.py        # Python tools: pycmplot, gwaslab, qmplot
   |--bench_r.r              # R tools: CMplot, qqman
   |--collect_results.py     # Aggregate CSVs, print summary tables, make figures
   |--summary.csv            # Summary of becnhmark results generated by collect_results.py
   |--pycmplot_job.sbatch    # SLURM submission for pycmplot
   |--gwaslab_job.sbatch     # SLURM submission for gwaslab
   |--qmplot_job.sbatch      # SLURM submission for qmplot
   |--cmplot_job.sbatch      # SLURM submission for CMplot
   |--qqman_job.sbatch       # SLURM submission for qqman
   |--figures/               # Generated plot images
```

## Step-by-step

### 0. Conda environments

Create two conda environments; one for Python tools, one for R:

```bash
# Python tools
conda create -n gwas_bench_py python=3.12.13
conda activate gwas_bench_py
pip install pycmplot gwaslab qmplot pandas numpy matplotlib

# or from included yml file (uncomment below)
#conda env create -n gwas_bench_py -f gwas_bench_py.yml

# R tools
conda create -n gwas_bench_r r-base=4.4.0
conda activate gwas_bench_r
Rscript -e "install.packages(c('CMplot','qqman','data.table','optparse'), repos='https://cloud.r-project.org')"

# or from included yml file (uncomment below)
#conda env create -n gwas_bench_r -f gwas_bench_r.yml
```

### 1. Generate synthetic data

```bash
conda activate gwas_bench_py
python generate_sumstats.py --all --outdir data/
```

This creates five files: `sumstats_500K.tsv` through `sumstats_10M.tsv`.
Seed is fixed (42) so data is identical across all runs.

### 2. Edit sbatch submission scripts
At minimum, update:

* `module load anaconda`: manner of loading anaconda on your cluster
* `--partition`: your cluster's partition name
* `-w`: the worker node to submit to
* `--time`: allow plenty of headroom for CMplot at 10M (can be very slow)

### 3. Submit
```
sbatch pycmplot_job.sbatch
sbatch gwaslab_job.sbatch
sbatch qmplot_job.sbatch
sbatch cmplot_job.sbatch
sbatch qqman_job.sbatch
```

### 4. Collect results

Once all jobs complete:

```
conda activate gwas_bench_py
python collect_results.py --resultsdir results/ --out summary.csv --plot
```
